## VS Code 插件

### 介绍

RTDF 提供了一个 VS Code 插件 RTDF for VS Code，用于在使用 RTDF 开发时提供一些便利。

### 安装

在 VS Code 插件中搜索 `RTDF for VS Code` 点击安装。

### 功能

光标悬浮于 RTDF 组件名称上时悬浮显示。

- 此组件 API。
- 直接打开 RTDF 官网此组件的示例、 API、 指南。
- 直接打开 RTDF 此组件源码。

### 配置

默认显示的 API 为简体中文，可到 VS Code 的设置页开启英文 API。
