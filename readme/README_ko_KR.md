# 소개

[React](https://react.dev), [Tailwind CSS](https://tailwindcss.com), any-tdf 디자인 시스템을 기반으로 한 모바일 Web 컴포넌트 라이브러리입니다.

> Ready | Tiny | Design | Fast

RTDF 는 React 에서 가능한 한 STDF 의 컴포넌트 설계와 API 형태를 따릅니다. Svelte 전용 기능은 React props, React nodes, render functions, Provider-based configuration 으로 매핑됩니다.

# 특징

- React 18/19, Tailwind CSS v4, TypeScript 를 완전히 지원합니다.
- 범용, 레이아웃, 내비게이션, 데이터 입력, 정보 표시, 피드백을 위한 모바일 우선 컴포넌트를 제공합니다.
- 컴포넌트 동작, 스타일, 아이콘, 애니메이션, 상호작용을 조정할 수 있는 풍부한 API 를 제공합니다.
- 다크 모드, 런타임 테마 전환, 사용자 정의 테마 생성, 42 개의 any-tdf 공유 테마를 지원합니다.
- 중국어와 영어 문서, 컴포넌트 예제, API 표, FAQ, 버전 노트를 제공합니다.
- 60 개 이상의 내장 로케일로 국제화를 지원합니다.
- 컴포넌트, 타입, 테마, 로케일, 유틸리티를 위한 전용 서브패스를 제공합니다.
- create-rtdf 로 Vite, React, Tailwind CSS v4, TypeScript, 아이콘, 테마 프리셋 프로젝트를 빠르게 생성할 수 있습니다.
- @any-tdf/vite-plugin-svg-symbol 기반 SVG Symbol 도구와 선택적 Iconify 지원을 제공합니다.
- VS Code 확장과 AI Skill 패키지로 RTDF 개발 효율을 높입니다.
