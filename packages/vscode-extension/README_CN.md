# 介绍

一个 VS Code 插件，用于在使用 [RTDF](https://rtdf.dev) 开发时提供组件 API 悬浮提示。

# 安装

在 VS Code 插件中搜索 `RTDF for VS Code` 点击安装。

# 功能

光标悬浮于 RTDF 组件名称上时显示：

- 当前项目 RTDF 版本与最新版本。
- 组件 API 文档。
- RTDF 官网示例、API、指南链接。
- 组件源码链接。

# 配置

默认显示简体中文 API，可在 VS Code 设置中开启 `RTDF.English` 显示英文 API。
