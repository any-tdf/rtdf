# Introduzione

Libreria di componenti web mobile basata su [React](https://react.dev), [Tailwind CSS](https://tailwindcss.com) e sul design system any-tdf.

> Ready | Tiny | Design | Fast

RTDF segue il design dei componenti e la forma delle API di STDF il più fedelmente possibile in React. Le funzionalità specifiche di Svelte sono mappate su React props, React nodes, render functions e configurazione basata su Provider.

# Caratteristiche

- Supporto completo per React 18/19, Tailwind CSS v4 e TypeScript.
- Set di componenti mobile-first per controlli generali, layout, navigazione, input dati, visualizzazione e feedback.
- API ricche per configurare comportamento, stili, icone, animazioni e interazioni.
- Modalità scura, cambio tema a runtime, generazione di temi personalizzati e 42 temi condivisi any-tdf.
- Documentazione in cinese e inglese, esempi di componenti, tabelle API, FAQ e note di versione.
- Internazionalizzazione con oltre 60 locale integrate.
- Export dedicati per componenti, tipi, temi, locale e utility.
- Scaffolding con create-rtdf per Vite, React, Tailwind CSS v4, TypeScript, icone e preset di tema.
- Strumenti SVG Symbol tramite @any-tdf/vite-plugin-svg-symbol e supporto opzionale per Iconify.
- Estensione VS Code e pacchetto AI Skill per accelerare lo sviluppo con RTDF.
