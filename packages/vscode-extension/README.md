# Introduction

A VS Code extension that provides component API hover docs for [RTDF](https://rtdf.dev) development.

# Installation

Search for `RTDF for VS Code` in the VS Code marketplace and install it.

# Features

Hover over an RTDF component name to view:

- Current and latest RTDF versions.
- Component API docs.
- RTDF website links for demos, API, and guides.
- Component source code link.

# Configuration

The default API language is Simplified Chinese. Enable `RTDF.English` in VS Code settings to show English API docs.
