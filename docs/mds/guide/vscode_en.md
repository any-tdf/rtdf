## VS Code Plugin

### Introduction

RTDF provides a VS Code plugin called RTDF for VS Code, which provides convenience when developing with RTDF.

### Installation

Search for `RTDF for VS Code` in the VS Code plugin marketplace and click install.

### Features

When hovering over the name of an RTDF component, the following information will be displayed:

- The API of this component.
- The example, API, and guide of this component on the RTDF site.
- The source code of this component.

The default API display is in Simplified Chinese. You can enable English API in the VS Code settings.
